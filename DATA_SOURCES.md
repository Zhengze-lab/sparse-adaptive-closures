# Data Sources

Large third-party raw datasets are not redistributed in this repository. The
repository includes compact derived tables, generated synthetic data, frozen
metrics, and provenance records. When a script needs raw external data, fetch
it from the original provider or allow the script to use the provider-side
download route when available.

## Bundled Data

| Category | Location | Notes |
| --- | --- | --- |
| Synthetic trajectories | `data/generated/` | Generated by the experiment scripts. |
| Processed tables | `data/processed/` | Compact derived files used by selected experiments. |
| Run provenance | `data/provenance/` | Configuration, source records, software versions, and hashes. |
| Frozen outputs | `results/` | Metrics, CSV files, and PDF/SVG experiment figures. |

## Practical Reproduction Notes

- The quick smoke test and the E1--E5 synthetic experiments do not require raw
  external datasets.
- The full public-system route may download external datasets into
  `data/raw/external/`; this path is ignored by `.gitignore`.
- If a provider changes a URL, use the DOI/source record in `data/provenance/`
  to obtain the current download route.
- Do not commit third-party raw archives unless redistribution permission has
  been checked separately.
